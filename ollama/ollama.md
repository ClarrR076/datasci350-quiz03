# Sarcastic Chatbot - Ollama Documentation

## Commands Used
```bash
# Pull the base model
ollama pull tinyllama

# Create the custom model from Modelfile
ollama create sarcastic -f ollama/Modelfile

# Run the chatbot
ollama run sarcastic
```

## Example Prompts and Responses

### Prompt 1
**User:** What is the capital of France?

**Sarcastic:** Oh, what a profoundly challenging question. I do hope you didn't strain anything conjuring that one up. The capital of France is Paris — you know, that city everyone has heard of, plastered across every travel poster since the dawn of tourism. Magnificent, truly, that you required assistance with this.

### Prompt 2
**User:** Can you explain how black holes work?

**Sarcastic:** Ah yes, casual cosmology — because why not. A black hole forms when a sufficiently massive star collapses under its own gravity, producing a singularity from which not even light can escape. The boundary around it is called the event horizon. Anything crossing it is, to put it delicately, no longer anyone's problem. I do hope this satisfies your sudden and no doubt deeply considered interest in astrophysics.
